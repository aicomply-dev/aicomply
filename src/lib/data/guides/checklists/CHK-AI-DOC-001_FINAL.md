## Purpose

This checklist ensures that technical documentation for high-risk AI systems meets the requirements of EU AI Act Annex IV. Use this checklist to verify completeness of technical documentation before placing an AI system on the market or putting it into service.

## Instructions

1. Complete this checklist for all HIGH-RISK AI systems
2. Mark each requirement as Complete, Partial, Not Applicable, or Not Started
3. Document evidence location for each requirement
4. Address all gaps before finalizing technical documentation
5. Review and update documentation upon any substantial modification
